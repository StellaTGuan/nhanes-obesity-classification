# nhanes-obesity-classification
Econ 695 project: predicting adult BMI category from NHANES 2021–2023 using logistic regression and boosting

## File Structure
- `data/` – raw and cleaned NHANES files (raw not committed)
- `R/` – scripts for download, cleaning, modeling, and evaluation
- `analysis/` – main R Markdown report
- `output/` – figures and tables for the report
